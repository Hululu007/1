## 食用方法 🍖
1. 复制下方生成的订阅链接，并将【验证码】替换为正确答案
2. 将正确的链接导入代理软件
3. 测速，选择一个【有值】的且延时较低的节点连接即可

## 破译方法 🎯
1. 可以先不更改【第一个】验证码，不断去尝试更改【第二个】验证码
2. 如果首轮验证失败就更改【第一个】验证码，再重复上述所说的步骤
